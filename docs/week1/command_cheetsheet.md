## 📋 1주차 Linux 명령어 치트시트

### 📂 파일 & 디렉토리 탐색

| 커맨드    | 설명                  |
| --------- | --------------------- |
| `pwd`     | 현재 위치 확인        |
| `ls`      | 현재 디렉토리 목록    |
| `ls -l`   | 상세 정보 포함 목록   |
| `ls -a`   | 숨김 파일 포함 목록   |
| `ls -lh`  | 파일 크기 보기 좋게   |
| `ls -la`  | 상세 + 숨김 파일 모두 |
| `cd 경로` | 디렉토리 이동         |
| `cd ~`    | 홈 디렉토리로 이동    |
| `cd ..`   | 상위 디렉토리로 이동  |

### 📄 파일 조작

| 커맨드                  | 설명                          |
| ----------------------- | ----------------------------- |
| `touch 파일명`          | 빈 파일 생성                  |
| `mkdir 디렉토리명`      | 디렉토리 생성                 |
| `mkdir -p 경로`         | 중간 디렉토리까지 한번에 생성 |
| `cp 파일 목적지`        | 파일 복사                     |
| `cp -r 디렉토리 목적지` | 디렉토리 복사                 |
| `mv 파일 목적지`        | 파일 이동 or 이름 변경        |
| `rm 파일명`             | 파일 삭제                     |
| `rm -f 파일명`          | 강제 삭제                     |
| `rm -rf 디렉토리`       | 디렉토리 강제 삭제 (주의!)    |

### 📖 파일 내용 확인

| 커맨드                    | 설명                        |
| ------------------------- | --------------------------- |
| `cat 파일명`              | 파일 내용 전체 출력         |
| `cat 파일A 파일B`         | 두 파일 연속 출력           |
| `cat 파일A 파일B > 파일C` | 두 파일 합쳐서 저장         |
| `less 파일명`             | 페이지 단위로 출력          |
| `more 파일명`             | 페이지 단위로 출력          |
| `echo "내용" > 파일명`    | 파일에 내용 작성 (덮어쓰기) |
| `echo "내용" >> 파일명`   | 파일에 내용 추가            |

### 🔐 권한 관리

| 커맨드              | 설명                  |
| ------------------- | --------------------- |
| `chmod 755 파일명`  | 권한 변경 (숫자 방식) |
| `chmod +x 파일명`   | 실행 권한 부여        |
| `chown 유저 파일명` | 소유자 변경           |
| `useradd 유저명`    | 새 사용자 생성        |

> 권한 숫자 계산
>
> | 숫자 | 권한                 |
> | ---- | -------------------- |
> | 7    | rwx (읽기+쓰기+실행) |
> | 6    | rw- (읽기+쓰기)      |
> | 5    | r-x (읽기+실행)      |
> | 4    | r-- (읽기만)         |
> | 0    | --- (권한 없음)      |

### 📦 apt 패키지 관리

| 커맨드                    | 설명                        |
| ------------------------- | --------------------------- |
| `apt update`              | 패키지 목록 최신화          |
| `apt upgrade -y`          | 설치된 패키지 업그레이드    |
| `apt install 패키지명 -y` | 패키지 설치                 |
| `apt remove 패키지명 -y`  | 패키지 삭제                 |
| `apt purge 패키지명 -y`   | 패키지 + 설정파일 완전 삭제 |
| `apt autoremove -y`       | 불필요한 패키지 정리        |
| `apt search 패키지명`     | 패키지 검색                 |
| `apt show 패키지명`       | 패키지 상세 정보            |

### ⚙️ 프로세스 관리

| 커맨드        | 설명                      |
| ------------- | ------------------------- | -------------------- |
| `ps`          | 현재 터미널 프로세스 확인 |

### 🔑 SSH

| 커맨드                              | 설명                    |
| ----------------------------------- | ----------------------- |
| `ssh-keygen -t rsa -b 4096`         | RSA 4096bit 키 생성     |
| `ssh -i 키파일.pem user@IP`         | pem 키로 서버 접속      |
| `chmod 700 ~/.ssh`                  | .ssh 디렉토리 권한 설정 |
| `chmod 600 ~/.ssh/id_rsa`           | 개인키 권한 설정        |
| `cat id_rsa.pub >> authorized_keys` | 공개키 등록             |

### 🔁 Bash 스크립트

| 문법                          | 설명                     |
| ----------------------------- | ------------------------ |
| `NAME="값"`                   | 변수 선언                |

| `for i in 1 2 3; do ... done` | for 반복문               |
| `while [ 조건 ]; do ... done` | while 반복문             |
